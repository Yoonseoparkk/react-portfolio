# 👩🏻‍🏫 React 포트폴리오 프로젝트 (React + TypeScript + Bootstrap)

## 💡 프로젝트 소개  
이 프로젝트는 **React + TypeScript + Bootstrap**을 사용한 간단한 **포트폴리오 프로젝트**입니다.  
추후 **Spring Boot Backend**와 연결 통한 API 연동 및 AWS 배포도 진행할 예정입니다. 🚀  

## 🗝️ 주요 기능  
✅ 자기소개 및 프로필 
✅ 프로젝트 열람  
✅ 방문자 방명록  
✅ Bootstrap을 활용한 UI 디자인
